# Running the Notebook in Google Colab

## Quick Start Steps

### 1. Upload Your Dataset to Colab

You have two options:

**Option A: Upload directly in Colab**
1. Open the notebook in Google Colab
2. Run the first cell which will create a file upload widget
3. Upload your `combined_dataset_clean.json` file

**Option B: Upload to Google Drive**
1. Upload `combined_dataset_clean.json` to your Google Drive
2. Mount Google Drive in Colab
3. The notebook will automatically find it

### 2. Open the Notebook in Colab

1. Go to [Google Colab](https://colab.research.google.com/)
2. Click "File" → "Upload notebook"
3. Upload `advanced_training_colab.ipynb`
4. Or create a new notebook and copy the cells

### 3. Run All Cells

Click "Runtime" → "Run all" or run cells one by one with Shift+Enter

## What's Different in Colab Version?

- ✅ Automatic dependency installation
- ✅ File upload widget for dataset
- ✅ Paths adjusted for Colab environment
- ✅ All project utilities included in notebook
- ✅ Models saved to Colab (download after training)

## After Training

1. Download the trained models from Colab's file browser
2. Or use the download code cell at the end
3. Models will be in `.pkl` format ready to use
